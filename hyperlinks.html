A hyperlink is represented by the html tag <a>. Hyperlinks tag is also called the anchor tag.
The body of a hyperlink is considered to be the area located between <a> and <a/> tags.
Any text placed inside the body of a hyperlink can be used to locate it with Selenium.
A hyperlink can be used to point to any webpage.
An anchor tag can also be used to point to another section in the same page.
It is also possible to invoke JavaScript when an anchor is clicked.
The attribute href is used for specifying the URL which will be opened when the link is clicked.
Selenium provides us with specific locators for finding links on a page such as locating hyperlink by text.
An anchor tag can also have other useful html attributes such as id, title, and target.
The target attribute of an href decides which if a new tab will be opened by clicking the link, new window will be opened, or the same tab will be used.
