What is a Form in HTML

HTML forms allow us to create user interface for taking input from users.
An HTML form can contain multiple input elements.
An HTML form can have a submit button. It is also possible to submit the form using any of the text input elements.
When a form is submitted, the data housed by the input fields is sent to a server.
It is possible to perform validation when a form is submitted.
Validation is essential to check if the user has provided input data in correct format or not.
Please never rely on client side validation solely. It is very easy to disable client side security checks.
Any malicious actor can bypass client side validation and transmit data to your server.
The server must have defensive logic to stop bad data crashing your application.
